# Svelte-simpleicons

## Requirement

TailwindCSS

## Installation

```sh
npm i -D @codewithshin/svelte-simpleicons
```

## Import

```html
<script>
  import { FacebookIcon } from '@codewithshin/svelte-simpleicons'
</script>

<FacebookIcon />
```

## Props

| Name      | Default   |
| --------- | --------- |
| className | h-6 w-6   |
| viewBox   | 0 0 24 24 |
